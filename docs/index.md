## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

# EMPC Demo Backstage OpenAPI

## Overview


That's the Software Template using Serverless Framework on AWS.
This Python application gets all data from a DynamoDB table to prove this simple architecture.
Some configuration is required for work the first time this code.
